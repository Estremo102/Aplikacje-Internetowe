Zadanie 3

Funkcje mogą również przyjmować nieokreśloną ilość zmiennych, można je odczytać z właściwości arguments, która działa podobnie do tablicy

function sum() {
    let total = 0;
    for(let i = 0; i < arguments.length; i++) {
        total += arguments[i];
    }
    return total;
}

Jednak dzisiaj zalecane jest używanie rest operator który zbiera dane do tablicy. aby z niego zkorzystać należy poprzedzić nazwę argumentu trzema kropkami

function sum(...numbers) {
    let total = 0;
    for(let i = 0; i < numbers.length; i++) {
        total += numbers[i];
    }
    return total;
}

Zadanie 4

Jeśli funkcja wymaga wartości i nie zostanie ona podana podczas jej wywołania będzie mieć wartość undefined, możemy jednak zmienić wartość domyślną używając operatora przypisania

function greet(name = "Gość") {
    return "Witaj, " + name + "!";
}

greet("Jan"); //Witaj Jan!
greet(); //Witaj Gość!

Zadanie 5

Funkcja może odwoływać się do samej siebie. Takie coś nazywamy rekurencją.

function fibonacci(x){
    if(x <= 0) {
        return 0;
    }
    if(x == 1 || x == 2) {
        return 1;
    }
    return fibonacci(x - 1) + fibonacci(x - 2);
}
